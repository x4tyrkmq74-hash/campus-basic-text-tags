Introduction to Semantic HTML

Semantic HTML means using tags that describe their meaning, which helps both browsers and assistive technologies understand the structure of a page.

For example, this paragraph uses emphasis to call attention to specific words and strong importance when something is extra important.

Developing good habits with semantic tags now will make your future projects easier to maintain, more accessible, and better for search engines.

Inline Text-Level Elements

Comparing Visual and Semantic Tags

You can use bold text with the <b> tag or italic text with the <i> tag, but it is often better to use <strong> and <em> when you want to add meaning, not just style.

Many tutorials, such as HTML Basics in zyBooks, will show you how to combine tags like <header>, <main>, and <footer> to create a clear document outline.
